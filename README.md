# Tour Bud

A mobile travel planning app that helps users manage trips, track budgets, store travel memories, and organize travel plans.

## Features
- Trip management
- Location planning
- Budget tracking
- Trip checklist
- Photo gallery
- Currency converter

## Tech Stack
Frontend: Flutter
Backend: Flask
Database: SQLite
Design: Figma
